# Keyword strategy

- Follow the keyword policy defined during `wp-onboarding` if one exists.
- Choose one main keyword or phrase that matches the real topic of the article.
- Add secondary phrases only when they fit naturally.
- Prefer search intent alignment over keyword density.
- Do not force keywords into every heading or paragraph.

# Article structure

- The title should contain the main phrase when natural.
- The slug should be short, readable, and topic-specific.
- The meta description should explain the value of the article in plain language.
- Headings should help both readers and scanning, not just ranking.
- Use internal linking where it genuinely helps the reader.

# On-page rules

- Use bold text sparingly to emphasize important terms, not for decoration.
- Avoid duplicate headings with tiny wording changes.
- Keep excerpts and meta descriptions distinct from each other when possible.
- If the article includes data, method, or a comparison, surface that in the title or excerpt when it improves clarity.

# Final note

- Valuable, specific writing usually beats keyword-heavy copy.
- SEO should strengthen the article, not distort it.
