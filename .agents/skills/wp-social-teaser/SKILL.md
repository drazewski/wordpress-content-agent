# Purpose

Use this skill only when the project context or user asks for teaser copy.

# Inputs

- finished article from `articles/`,
- `project-context.md`,
- optional platform notes,
- optional target URL.

# Outputs

Prepare only the formats the user requested. Possible outputs:

- homepage teaser,
- short social post,
- longer social post,
- meta-style share summary,
- suggested image direction,
- optional CTA.

# Rules

- Keep the teaser faithful to the article.
- Do not exaggerate findings or promise more than the article delivers.
- Avoid clickbait and vague curiosity hooks.
- Match the tone and audience from `project-context.md`.
- If the article is based on research, preserve uncertainty and limits.
