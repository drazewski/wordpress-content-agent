# Tags Map

This is a starter map. Replace it with the real taxonomy during onboarding when the site already has categories and tags.

## Generic Categories

- Analysis
- Guides
- Research
- News
- Reviews
- Opinion

## Generic Tag Patterns

Use concrete tags for:

- main topic,
- place,
- named organization,
- method or technology,
- article format,
- species, product, framework, or system when relevant.

Avoid tags that only describe the existence of an article, such as:

- article,
- post,
- update,
- topic,
- interesting.
